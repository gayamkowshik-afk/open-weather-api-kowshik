Real-world consequences of exposing an API key on GitHub

Exposed API keys can be used by unauthorized users, leading to unexpected charges and quota exhaustion. Attackers may misuse the service, causing downtime or degraded performance. It can also result in data exposure and potential account suspension by the API provider.

Why the company’s privacy policy prohibits logging city names

City names are considered location data and may be personally identifiable when combined with other information. Avoiding such logs follows data-minimization principles and reduces privacy risks. It also helps prevent tracking user behavior and limits damage in case of data breaches.
